Mis desafíos

Construyendo un landing page

Se construyó un landing page del producto preferido. En este caso, una tienda ficticia de consolas. Para lograrlo, se implementó CSS utilizando archivos de hojas de estilo externas al documento HTML y de base para homogeneizar el diseño del sitio.

Iguana page

Se aplicaron los nuevos conocimientos de posicionamiento en CSS. Se combinó lo aprendido de flex con las unidades anteriores. Se solicitó construir un landing page a partir de una maqueta, siguiendo las especificaciones definidas dentro de la misma.

Cuppon

Se implementó el sistema de grillas de Bootstrap, distribuyendo el contenido adecuadamente, para hacer una página web adaptable a distintos dispositivos, aplicando las clases de Bootstrap a través de los componentes card y formularios.
